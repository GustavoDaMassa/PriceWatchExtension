Política de Privacidade

Última atualização: maio de 2026

A extensão PriceWatch — Monitor de Preços ("a extensão") é desenvolvida e mantida por Gustavo Henrique. Esta política explica quais dados são coletados, como são usados e como são protegidos.

1. Dados coletados

  • Credenciais de acesso: e-mail e senha fornecidos voluntariamente no popup da extensão para autenticação na plataforma PriceWatch.
  • Token de autenticação (JWT): gerado após o login bem-sucedido e armazenado localmente no dispositivo do usuário via chrome.storage.local.
  • URLs de produtos: a URL canônica da página do Mercado Livre visitada pelo usuário, enviada à API do PriceWatch somente quando o botão "Acompanhar" é acionado.

2. Como os dados são usados

  • As credenciais são transmitidas à API do PriceWatch exclusivamente para autenticação. Não são armazenadas localmente além do token JWT resultante.
  • O token JWT é usado para autorizar as chamadas à API e é mantido apenas no armazenamento local do Chrome do próprio usuário.
  • As URLs de produtos são enviadas à API para cadastro e monitoramento de preços, conforme solicitado explicitamente pelo usuário.

3. Compartilhamento de dados

Nenhum dado é vendido, alugado ou compartilhado com terceiros. Os dados trafegam exclusivamente entre o dispositivo do usuário e a API do PriceWatch (pricewatch-api.gustavohdev.com.br).

4. Armazenamento e segurança

  • O token JWT é armazenado em chrome.storage.local, acessível apenas pela própria extensão.
  • Toda comunicação com a API utiliza HTTPS.
  • O token tem validade limitada e é removido automaticamente ao expirar ou ao usuário fazer logout.

5. Direitos do usuário

O usuário pode remover todos os dados armazenados pela extensão a qualquer momento clicando em "Sair" no popup ou desinstalando a extensão.
